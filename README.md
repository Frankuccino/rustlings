## Purpose

The goal of this repository is to reinforce Rust fundamentals by solving short exercises that focus on core language concepts such as ownership, borrowing, error handling, and Rust syntax.

Rustlings provides immediate feedback, allowing for rapid iteration and learning through debugging and compiler messages.

## What I'm Practicing

The exercises cover key Rust concepts including:

- Variables and mutability
- Functions
- Control flow (`if`, `loop`, `while`, `for`)
- Ownership
- Borrowing and references
- Structs
- Enums
- Error handling (`Option<T>`, `Result<T, E>`)
- Collections
- Traits and generics
- Tests and modules

## Workflow

My typical learning workflow:

1. Attempt the exercise without looking at the solution.
2. Use compiler errors and hints to debug.
3. Fix the implementation until the exercise passes.
4. Commit progress after completing a section.

## Running Rustlings

To run the exercises locally:

```bash
cargo install rustlings
rustlings init
cd rustlings
rustlings watch
```

The `watch` command automatically checks exercises and moves to the next one once the current exercise passes.
